## Project Description

一个改装毛绒玩具，具有功能：

1. 被人摔打（受力）后，有语言反馈，起到发泄作用
2. 可以根据受力大小和类型不同，发出不同的语音
3. 外表滑稽有趣，质地柔软，摔不坏
4. 可以连接网络，根据流行语自动更新语音库
5. 在闲置时间可以作为微信的终端，自动播放收到语音

## How to Make it

基于Arduino开发板，在现有玩偶的基础上改装，利用力传感器、陀螺仪，反馈玩具的受力状态，根据参数开发语音库。
